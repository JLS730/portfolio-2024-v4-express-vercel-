Pawsitively Yours

Pawsitively Yours is dog adoption created using React, Firebase, Sass, React Router and Postman to fetch data from the Petfinder API. Users are able to create an account, browse and search dogs, and assign pets as favorites for later viewing. Users are also able to save and manage their personal information that is stored on the firestore database.

DeepBlue Pursuits

DeepBlue Pursuits is an online fishing rod shop created using React, Firebase, Sass, React Router and Paypal to handle payments. Users are able to create an account, add and remove items from cart, and add user information for autofill on checkout. User cart is updated using firestore database.
